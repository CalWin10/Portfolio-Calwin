## 📖 About

A modern, animated portfolio website showcasing my work as a **Data Engineering student** and **Full-Stack Developer**. Built with React, Styled Components, and Framer Motion for smooth animations.

## ✨ Features

- 🎨 **Interactive Animations** - Smooth page transitions and floating elements
- 🌓 **Dark/Light Theme** - Dynamic theme switching with particle backgrounds
- 📱 **Fully Responsive** - Works seamlessly on all devices
- 🎵 **Background Music** (Optional) - Toggle audio player
- 📝 **Blog Section** - Showcase achievements and hackathon wins
- 💼 **Projects Gallery** - Flip cards displaying my work
- 🧠 **Skills Section** - Data Engineering & Full-Stack expertise
- 📧 **Contact Integration** - Direct email link via Outlook
